<h3>Prerequisites</h3>
<p>Backend</p>
<li>Rails 5</li>
<li>ActiveModelSerializers</li>
<li>ActionMailer</li>
<li>PostgreSQL</li>
<li>JWT</li>
<li>BCrypt</li>

<h3>Installing</h3>
<h7>Clone the backend repo and run the rails server by:</h7>

```
$git clone git@github.com:elikantor/Survey-Genie-Backend.git
$cd Survey-Genie-Backend
$bundle install
$rails db:create
$rails db:migrate
$rails db:seed
$rails s
```
